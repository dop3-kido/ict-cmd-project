## 25 Command Prompt Commands

1. dir – Displays files and folders  
2. cd – Changes directory  
3. cls – Clears the screen  
4. date – Shows date  
5. time – Shows time  

6. copy – Copies files  
7. move – Moves files  
8. del – Deletes files  
9. ren – Renames files  
10. mkdir – Creates folder  

11. rmdir – Deletes folder  
12. type – Shows file content  
13. echo – Displays text  
14. pause – Pauses execution  
15. exit – Closes CMD  

16. help – Shows commands  
17. start – Opens program  
18. tree – Shows folder structure  
19. attrib – File attributes  
20. color – Changes color  

21. hostname – Shows computer name  
22. ipconfig – Shows IP address  
23. ping – Tests network  
24. shutdown – Shuts down PC  
25. tasklist – Shows running programs
